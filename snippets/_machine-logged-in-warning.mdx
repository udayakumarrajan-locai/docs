<Warning>

    This flow can be executed using a machine user.

    - Send a valid `Authorization` header with of type `Bearer TOKEN_VALUE` where **TOKEN_VALUE** is the a JWT returned in the response from [Machine User Token](/api-tutorials/unauthenticated-flows/generate-machine-user-token) flow
    - In case your app supports cookie, you can enable request cookies as our API Automatically sets a cookie named `session-token` which is forwarded with each request to our api.
</Warning>
